How it works

Listing → Verification → Classification → Clearing → Settlement

v1 is an operational pipeline. It produces artifacts: an Asset ID, a verified profile, a class assignment, and a registry entry.

01
List
Submit territory/asset parameters and initial documentation.
Output: Asset ID
02
Verify
Route to scientific verification layer (institutional workflow).
Output: Verified profile
03
Classify
Assign capital class + reference band according to the Exchange taxonomy.
Output: Class + band
04
Clear
Register into the clearing ledger and lock reserve state.
Output: Registry entry
05
Settle
Institutional settlement via approved counterparties and terms.
Output: Agreement / instrument
